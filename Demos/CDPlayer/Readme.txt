CD Player
_________

Программа является примером использования следующих компонентов и 
утилит Extension Library:
  TELTrayIcon
  TELInstanceChecker
  ELPackStrings
  ELUnpackStrings

Программа представляет собой проигрыватель аудио  
компакт дисков со следующими дополнительными возможностями:
  управление из командной строки;
  регистрация себя как проигрывателя Windows по умолчанию;
  выключение компьютера по окончании проигрывания диска.


Управление из командной строки
______________________________

В главном меню Windows выберите пункт "Выполнить" ("Run") и
введите команду в формате описанном ниже.

Программа поддерживает следующие ключи командной строки:
  play
  stop
  prevtrack
  nexttrack
  closedoor
  opendoor

Например:
  ...\CDPlayer.exe play

В случае использования ключа "play" дополнительно может быть 
указано имя файла для воспроизведения (*.cda файл), например:
  ...\CDPlayer.exe play "D:\Track03.cda"


Регистрация программы как проигрывателя Windows по умолчанию
____________________________________________________________

После регистрации программы она будет автоматически запускаться 
при:
  вставке аудио компакт диска в устройство чтения компакт дисков;
  двойном щелчке на иконке аудио компакт диска в проводнике Windows;
  двойном щелчке на иконке .cda файла или ярлыка к .cda файлу.

Для регистрации программы как проигрывателя Windows по умолчанию
в контекстном меню программы выберите пункт 
"Register as default player".

ВНИМАНИЕ! Некоторые проигрыватели, например WinAmp, проводят 
автоматическую перерегистрацию в реальном режиме времени.
По этому, если Вы используете такие проигрыватели, то перед 
регистрацией необходимо закрыть все такие программы или изменить
их настройки. В случае использования WinAmp необходимо также
закрыть WinAmp Agent.


Выключение компьютера по окончании проигрывания диска
_____________________________________________________

Программа имеет возможность выключения компьютера по окончании
проигрывания диска. Для активации этого режима необходимо в 
контекстном меню программы выбрать пункт "Power off mode".
Этот режим доступен только если воспроизведение компакт диска
уже запущено.


TELTrayIcon
___________

Компонент TELTrayIcon используется в программе для представления 
ее в System tray и индикации режима работы программы. Щелчок левой
кнопкой мыши на иконке программы показывает главное окно программы,
щелчок правой кнопкой - контекстное меню.

В Windows доступ к управлению устройством чтения компакт дисков
в определенный момент времени может иметь только одна программа.
По этому, если на момент старта программы уже запущен другой 
проигрыватель компакт дисков, то программа переключается в режим 
ожидания освобождения устройства чтения компакт дисков.
Этому режиму соответсвует мигающая иконка (использование 
возможности анимации компонента TELTrayIcon). Для создания такой 
ситуации запустите стандартный Windows проигрыватель компакт дисков.
Затем запустите программу. До тех пор пока Вы не закроете Windows
проигрыватель Вы будете видеть мигающую иконку.


TELInstanceChecker
__________________

Компонент используется для предотвращения возможности одновременной
работы более чем одной копии программы в системе и для передачи
параметров запуска одной копии программы другой.

При запуске программа при помощи этого компонента проверяет наличие
в системе уже работающей другой копии программы, если находит 
ее, то передает ей свои параметры запуска и заканчивает работу.
Таким образом организуется управление из командной строки и запуск
.cda файлов из проводника Windows.


ELPackStrings и ELUnpackStrings
_______________________________

Функции используются в программе для преобразования набора 
параметров запуска в строку (и обратно), которая может быть 
передана другой копии программы при помощи компонента 
TELInstanceChecker.
